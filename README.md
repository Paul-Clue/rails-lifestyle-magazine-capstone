## Getting Started

Follow these simple steps to get the app working.
### Prerequisites

Ruby: 2.6.3
Rails: 5.2.3
Postgres: 9.5

### Setup

Instal gems with:

```
bundle install
```

Setup database with:

```
   rails db:create
   rails db:migrate
   rails db:seed
```

### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    rpsec --format documentation
```
